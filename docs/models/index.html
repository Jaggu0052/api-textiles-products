Field types and validation summary.

Product

  • name: string (required) - validation: string min 1
  • description: string (optional)
  • price: number (required) - validation: nonnegative
  • imageUrl: string (optional) - validation: url
  • inStock: boolean (default true)

User

  • name: string (required) - validation: min 1
  • email: string (required) - validation: email
  • passwordHash: string (required) - validation: min 6
  • userTypeId: ObjectId (optional)

User Type

  • name: string (required, unique) - validation: min 1
  • description: string (optional)

Stock

  • productId: ObjectId (required) - validation: string min 1
  • quantity: number (required) - validation: int, nonnegative
  • location: string (optional)

Payment

  • orderId: ObjectId (required) - validation: string min 1
  • amount: number (required) - validation: nonnegative
  • paymentTypeId: ObjectId (optional)
  • status: enum (pending|paid|failed)
  • reference: string (optional)

Payment Type

  • name: string (required, unique) - validation: min 1
  • description: string (optional)

Cart

  • userId: ObjectId (required) - validation: string min 1
  • items: array of { productId, quantity }

Order

  • userId: ObjectId (required) - validation: string min 1
  • items: array of { productId, quantity, price }
  • totalAmount: number (required) - validation: nonnegative
  • status: enum (pending|paid|shipped|delivered|cancelled)

Favorite

  • userId: ObjectId (required) - validation: string min 1
  • productId: ObjectId (required) - validation: string min 1
